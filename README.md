# alu-back-end

This project contains Python scripts to interact with a REST API, gathering employee data and exporting it to various formats like CSV and JSON.

## Tasks
* `0-gather_data_from_an_API.py`: Gathers data from the API and prints the TODO list progress.
* `1-export_to_CSV.py`: Exports the TODO list to a CSV file.
* `2-export_to_JSON.py`: Exports the TODO list to a JSON file.
* `3-dictionary_of_list_of_dictionaries.py`: Exports all employees' tasks to a JSON file.
