# API Project

This directory contains Python scripts for interacting with a REST API to gather and export data.

## Files
- `0-gather_data_from_an_API.py`: Gathers employee TODO tasks
- `1-export_to_CSV.py`: Exports to CSV
- `2-export_to_JSON.py`: Exports to JSON
- `3-dictionary_of_list_of_dictionaries.py`: Exports all employees to JSON
